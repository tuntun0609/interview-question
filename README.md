# Interview Question

一个基于 Next.js 构建的现代面试题解答平台。

[![Netlify Status](https://api.netlify.com/api/v1/badges/bd9ce472-f5ec-4d3b-81a8-bc051a8fb8f7/deploy-status)](https://app.netlify.com/sites/fe-question/deploys)

## 技术栈

- **框架**: Next.js
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **数据库**: Drizzle ORM
- **认证**: Clerk
- **Markdown**: Contentlayer
- **UI组件**: shadcn/ui

## 主要功能

- 📝 面试题的展示和解答
- 🔍 按类别和难度搜索题目
- 👤 用户认证和权限管理
- 📱 响应式设计，支持移动端

## 开发

1. 安装依赖：
```bash
pnpm install
```

2. 启动开发服务器：
```bash
pnpm dev
```

3. 构建生产版本：
```bash
pnpm build
```
